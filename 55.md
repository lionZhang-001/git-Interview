.关于 channel 下面描述正确的是？

```
A. close() 可以用于只接收通道；

B. 单向通道可以转换为双向通道；

C. 不能在单向通道上做逆向操作（例如：只发送通道用于接收）
```

```
n答案：
Ｃ
关于A为啥是错误的，从它的官方库里面就可以找到答案．
close函数是一个内建函数， 用来关闭channel，这个channel要么是双向的， 要么是只写的（chan<- Type）。 
这个方法应该只由发送者调用， 而不是接收者。
```

```
type T struct {
	n int
}

func getT() T {
	return T{}
}

func main() {
	getT().n = 1
}
```

```
答案：
T{}无法获取到地址
```
